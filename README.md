## Features

- Type safe
- Unit tests
- Dark mode
- Translations
- Infinite Scrolling
- Responsive Layout
- Persistence of user settings

## Build Setup

```bash

From Node v14.21.1

# install dependencies
$ npm run install

# Developement mode with hot reload
$ npm run dev


```

## Running Tests

```npm run test:unit```
